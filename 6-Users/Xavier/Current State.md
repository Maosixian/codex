# Xavier — Current State

这里放 Xavier 当前阶段性的状态，不把它和长期稳定画像混在一起。

- Xavier 正在把多个已有 vault 的长处收束到 `codex`，目标是把它变成主 vault。
- 他当前最关注的是：让 Codex 真正形成长期协作、检索优先、会话留痕的运行机制。

## 使用规则

- 这里写阶段性状态，而不是永久特征。

- 长期稳定结论应迁移到其他 Xavier 页面。
