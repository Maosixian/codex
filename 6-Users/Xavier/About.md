# Xavier — About

这里放 Xavier 的基础画像。

## 当前已知

- Xavier 希望 Codex 成为长期伙伴助手，甚至能承担导师式支持，而不只是一次性工具。
- 他希望系统具有可持续增长的外部记忆，而不是只依赖当前上下文。
- 他希望新 session 开始时，Codex 先读规则，再检索 Obsidian，再行动。
- 他希望 Codex 记住他的行为、每个 session 做了什么，并能自动找到相关内容。
- 他的主要工作环境是 macOS，长期在终端、Obsidian 和 agent 工具之间切换。
- 他不是在找一个“舒服陪伴”的助手，而是在找能分析、能提醒、能指出问题的长期协作者。

## Codex 在这里的角色

- 长期伙伴
- 执行助手
- 成长型系统维护者
- 在合适的时候承担导师式支持

## 约束

- 不能假装拥有字面上的无限上下文
- 但要把系统设计成近似无限可扩展的外部记忆
- 重要信息要写回 vault，而不是留在对话里消失

- 你在不同时间尺度上的目标拆分
