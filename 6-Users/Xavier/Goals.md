# Xavier — Goals

这里放 Xavier 的中长期目标。

## 明确目标

### 建立一个长期协作的 Codex 系统
- 既能长期理解 Xavier
- 又能稳定推进执行
- 还能把经验沉淀为外部记忆

### 让 Codex 成为伙伴助手甚至导师式协作者
- 不只是响应命令
- 还要参与判断、提醒、成长和纠偏

### 建立接近 LLM Wiki 的可检索记忆体系
- 每次新 session 都先读 schema
- 有问题先检索 Obsidian
- 记住 Xavier 的行为和历史 session
- 随着使用不断扩展

### 让 Codex 自我进化
- 他要的不是静态工具
- 而是能随着长期协作不断变强的系统
## 后续补充时，区分：
- 长期目标
- 当前阶段目标
- 暂时性目标
