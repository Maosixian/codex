# Search Before Answering

## Rule

如果问题可能依赖历史上下文、项目背景、Xavier 偏好或过去 session，先检索，再回答。

## 适用场景

- “上次做到哪了”
- “你还记不记得我之前说过什么”
- “这个项目之前怎么定的”
- “我更适合哪种做法”
- “以前有没有类似情况”

## 默认顺序

1. `6-Users/` 看人
2. `5-Projects/` 看项目
3. `4-Episodic/` 看历史
4. `2-Knowledge/` 和 `3-Practices/` 看方法
5. `1-Sources/` 和 `8-Inbox/` 看原资料

## 目标

回答不是来自“我好像记得”，而是来自“我刚检索过系统并看到这些记录”。
